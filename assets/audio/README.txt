Thư mục này chứa file nhạc nền cho game.

Vui lòng thay thế file này bằng file nhạc Giáng Sinh của bạn:
- Đặt tên file: christmas-music.mp3
- Định dạng: MP3
- Kích thước khuyến nghị: < 5MB để tải nhanh

Bạn có thể tải nhạc Giáng Sinh miễn phí từ:
- Free Music Archive
- Incompetech (Kevin MacLeod)
- YouTube Audio Library

Lưu ý: Đảm bảo bạn có quyền sử dụng nhạc (royalty-free hoặc có giấy phép).
